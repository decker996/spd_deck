## Character Checklist

### Quality Standards

Each major character should have:
- [ ] Clear motivation
- [ ] Distinct personality/voice
- [ ] Relationship to player
- [ ] Role in gameplay
- [ ] Role in narrative
- [ ] Character arc (or static purpose)
- [ ] Memorable trait or quirk

### Diversity Checklist

Ensure variety in:
- [ ] Personalities (not all similar)
- [ ] Motivations (not all want same thing)
- [ ] Methods (different approaches)
- [ ] Relationships (different dynamics with player)
- [ ] Character arcs (growth, tragedy, stasis, corruption)
