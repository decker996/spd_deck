## Quality Checklist

### Scene Quality Standards

Each scene should:
- [ ] Advance plot, character, or theme
- [ ] Have a clear purpose
- [ ] Offer meaningful choice (if interactive)
- [ ] Use appropriate tone and voice
- [ ] Be the right length (not too long/short)
- [ ] Adapt to game state where appropriate
- [ ] Connect to other scenes coherently
